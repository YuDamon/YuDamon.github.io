模型评价的基本概念，ROC曲线，PR曲线，图像分类/检测/分割任务中的评价指标。

1 基本概念

True Positive：正确地预测为1，TP
True Negative：正确地预测为0， TN
False Positive : 错误地预测为1， FP，误警
False Negative：错误地预测为0， FN，漏警

Precision：精度，查准率
在预测值为1中，正确预测的比例
Precision = TP / P = TP / (TP + FP)

Recall : 召回率，查全率
在真实值为1中，正确预测的比例
Recall = TP / (TP + FN)

Accuracy
预测对的 / 总样本数
ACC = (TP+TN)/ all

真阳性率 TPR：分类器 “正确识别出的正实例” 占 “所有实际正实例” 的比例
TPR = TP / P = TP / (TP + FN)
也叫 灵敏度sensitivity

假阳性率 FPR：分类器 “错认为正类的负实例” 占 “所有实际负实例” 的比例
FPR = FP / N = FP / (FP + TN)
= (1-特异度)
特异度（specificity，也叫真阴性率TNR）

2 ROC曲线

receiver operating characteristic curve，
也叫感受性曲线（sensitivity curve）


上图的(b)图形化了TP，TN，FP，FN的概念：两个峰分别是样本的真实分布，分类模型以图中theta作为分类面。
横轴 FPR，纵轴TPR，类似于“以代价(costs<->FPR)对收益(benefits<->TPR)”的曲线，

ROC曲线下方的面积（Area Under the ROC Curve, AUC）用来评价模型平均性能，显然AUC越大性能越好。完美模型的AUC = 1，如简单的随机猜测模型，AUC = 0.5

3 PR曲线

Precision（纵坐标）-Recall（横坐标）曲线（查准率-查全率曲线）

对于一个分类器，通过调整分类阈值，得到不同的P-R值，即得到一条曲线。通常随着分类阈值从大到小变化，P减小，R增加
比较两个分类器好坏时，显然是查得又准又全的比较好，也就是的PR曲线越靠近坐标（1，1）的位置越好

4 ROC和PR曲线的联系

  • 绘制ROC曲线和PR曲线都是选定不同阈值，得到不同的x轴和y轴的值，画出曲线
  • 希望的直观效果：ROC曲线凸向左上方；PR曲线右上凸
    *作为衡量指标，选择ROC或PR都是可以的。ROC和PR虽然具有相同的出发点，但并不一定能得到相同的结论（与正负样本数量相关，比如：负样本很多的时候，两者就截然不同了，ROC效果依然看似很好，但是PR上反映效果一般）详见paper：Jesse Davis,Mark Goadrich. The Relationship Between Precision-Recall and ROC Curves.
  • 所以，有时候也只能参考着别人已有的进行选择了
