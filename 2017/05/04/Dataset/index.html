1 通用视觉数据集

MNIST
0-9的手写数字。训练样本60,000；测试样本10,000. The digits have been size-normalized and centered in a fixed-size image
现在一般用来简单地验证下算法
MNIST handwritten digit database

CIFAR 10 & CIFAR 100
图像分类/识别数据集
CIFAR-10 含60,000（共10类，每类有6000）个32x32的彩色图像，训练样本50,000，测试样本10,000。

CIFAR-100 和CIFAR-10类似，但有100类（20个大类superclasses），每类600个，500训练样本，100测试样本

CIFAR-10 and CIFAR-100 datasets

ImageNet
目前世界上最大的图像识别数据集，官网被墙mmp
Imagenet ILSVRC（ImageNet Large Scale Visual Recognition Challenge） 大规模视觉识别挑战赛包括“图像分类和物体检测”两项，从2010年开始每年举办一次

PASCAL VOC
05-12年每年举办挑战赛，目标检测和分割。
常用的有07和12两个版本


The PASCAL Visual Object Classes Homepage

MS COCO

也有挑战赛
Common Objects in Context

2 交通相关数据集

CityScapes

KITTI
车辆检测
7481 training images and 7518 test images, comprising a total of 80.256 labeled objects. All images are color and saved as png
KITTI-Object Detection Evaluation 2012

TME Motorway
实际路况下的车辆检测数据集，28段，不同场景总共30,000+帧

Toyota Motor Europe (TME) Motorway Dataset

CompCars
应该是目前最大的车辆数据集，需要申请。163 car makes with 1,716 car models, a total of 136,726 images.
The Comprehensive Cars (CompCars) dataset
