怎么折腾出来的

Hexo + GitHub Pages

Hexo是什么

一个开源的基于node.js的博客框架，简单高效，支持Markdown，插件丰富。
了解更多, 戳 Hexo介绍页 / GitHub项目

GitHub Pages又是什么

可编写的静态网页，托管在GitHub上，空间免费稳定。


以上，对于计算机相关专业的盆友来说：Aha, interesting.

其他骚年，就算从未接触过GitHub/命令行/前端/后台/Markdown…
也没关系，只要你想，两个小时以内绝对可以亲手搭建一个属于自己的博客

当写好一篇文章准备部署的时候
在Git Bash上敲出一行命令
重重地砸下Enter
黑框里不知所云的白色英文和数字不断向下跳动刷新的时候…
你会觉得
“啊！我好厉害，虽然根本不知道自己在干什么”

而你的室友
也将目瞪口呆

是不是很有煽动性啊哈哈哈
然后，你就会不自觉地花上n个小时去美化，去完善:
配置主题文件，集成各种第三方服务（评论、站点统计、分享功能哈哈哈…）

Hierarchy将有些什么

  • 学习笔记。
    包括但不限于IV, ADAS, CV, ML/DL, C/C++/Python/Matlab, Git…
    希望能督促自己写出些原创性/有价值的内容，这也是“Hierarchy”存在的最重要原因
  • 兴趣爱好相关（前沿科技/篮球/摄影/吉他/创新创业/军事)
  • 想和大家讨论交流的话题。
  • 分享其他有用或有趣的内容。

总的来说：记录 表达 分享 交流

希望能至少一周一更

Writing is about content, about what you want to say – not about fancy formatting.
